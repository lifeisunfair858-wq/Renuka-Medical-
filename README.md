# Renuka Medical - Pharmacy Inventory System

A complete pharmacy inventory management system with cloud sync and offline support.

## Features
- 📦 Product inventory management (CRUD)
- 🛒 Sales recording with invoice generation
- 📊 Sales history and reports
- ⏰ Expiry date alerts (30/60/90 days)
- 🗄️ Shelf organization view
- ☁️ Supabase cloud sync (works across devices)
- 📱 Offline mode with auto-sync queue
- 🤖 AI invoice scanning (Gemini)
- 📄 Export to Excel, PDF, JSON
- 📧 Email low stock alerts

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Supabase (Backend & Database)
- Gemini AI (Vision API)
- SheetJS (Excel export)
- html2pdf (PDF generation)
